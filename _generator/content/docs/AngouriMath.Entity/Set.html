  • Method Filter(AngouriMath.Entity,AngouriMath.Entity.Variable)

    Summary

    Adds a constraint to every element of a set. 1) For a finite set, it will add Provided for every element 2) For an interval and special set, it will wrap it with a cset (e. g. [a; b].Filter(x > 0, x) -> { x : x in [a; b] and x > 0 }) 3) For a cset, it will add a predicate (e. g. { y : y2 = 3 }.Filter(x > 0, x) -> { y : y2 = 3 and y > 0 })
