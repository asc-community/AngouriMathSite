  • Method RollBackToDefault

    Summary

    Use it in case if you need to cancel any adjustments of the setting and rollback to the one predefined in MathS. It does not roll back to the previous value. Instead, it sets the value which is defined right after MathS is initialized. If you need recoverable settings (aka local), use method As(`0,System.Action).

  • Method Set((`0)

    Summary

    Sets the new value for the setting

    Parameter value

    Returns

    An IDisposable struct. Make sure to use operator `using` before, so that it auto-disposes once the method is over. Example: using var _ = MathS.Settings.SomeSetting.Set(4); // do something, once the method is ended, the setting is automatically returned to the initial statement
