◆ ApproachFrom

enum AngouriMath.Core.ApproachFrom
strong

Where to tend to the given number in limits

Enumerator
BothSides 

Means that the limit is considered valid if and only if Left-sided limit exists and Right-sided limit exists and they are equal

Left 

If x tends from the left, i. e. it is never greater than the destination

Right 

If x tends from the right, i. e. it is never less than the destination
