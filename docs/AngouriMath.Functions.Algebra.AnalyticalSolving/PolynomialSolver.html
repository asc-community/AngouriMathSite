    Summary

    So that the final list of powers contains power = 0 and all powers >= 0 (e. g. if the dictionaty's keys are 3, 4, 6, the final answer will contain keys 0, 1, 3, if the dictionary's keys are -2, 0, 3, the final answer will contain keys 0, 2, 5)

    Dictionary to process. Key - power, value - coefficient of the corresponding term
